# 📈 Result Analysis

## 🔐 Result Analysis Access

Only an authorized employee can view the result analysis for internal exams in the Result Analysis section.

## 🎯 Pre‑Analysis Selection

Before you generate any analysis, you must first choose the relevant Program, Academic Year, and Institute from the available dropdowns or filters.

## 📊 Displayed Metrics

Once your selections are applied, the grid will show:

- Total number of students present in the exam
- Total number of students absent
- Number of students who passed
- Number of students who failed
- Pass percentage for the cohort

## 📋 Grid‑Style Layout

All of these key statistics appear together in a structured, grid‑format table, allowing you to quickly assess overall exam performance.
