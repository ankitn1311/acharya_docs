# 🌴 Vacation Leave Configuration

*(Clarification needed on certain features)*

The **Vacation Leave Configuration** module enables HR and administrators to define and manage vacation leave allocations for employees at specific institutes within a set timeframe.

---

## ✨ Key Features

1. 🏫 **Institute-Specific Leave Settings**  
   Set vacation leave allowances tailored to individual institutes.

2. 📅 **Custom Leave Periods**  
   Specify the **valid time period** during which vacation leave days can be utilized.

3. 👥 **Employee Leave Allocation**  
   Allocate a defined number of vacation leave days per employee, which can vary based on institute and academic year.
